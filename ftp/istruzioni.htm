Istruzioni per l'uso dell'Area di Lavoro

Cartella principale "web root"
Quando aprite l'area di lavoro vi trovate nella sua "cartella principale".
Corrisponde alla vostra "web root" ed � qui che dovete mettere il file della homepage, e tutti gli eventuali file e cartelle a cui l'homepage si riferisce.

Caricamento di file
Per caricare file nella cartella in cui vi trovate cliccate sull'icona "nuovo file" e quindi dal modulo che segue cliccate su "Sfoglia..." e individuate (come se voleste aprirlo) il file che intendete caricare. E' possibile caricare solo un file per volta.

Per caricare pi� di un file per volta, comprimete i file nel vostro computer usando WinZip e quindi caricate l'unico file .zip ottenuto cliccando sull'icona "archivio .zip" .
I file verranno automaticamente decompressi nella cartella corrente.

Cancellazione di file e cartelle
Per cancellare uno o pi� file dalla cartella in cui vi trovate "spuntate" i file che intendete cancellare e quindi cliccate sul pulsante "Cancella elementi selezionati".
Potete "spuntare" anche una cartella, ma per poter essere cancellata questa deve essere vuota (nel caso quindi dovrete prima entrare nella cartella e cancellarne il contenuto).

Nomi dei file
I nomi dei file HTML devono avere estensione .htm;
I nomi dei file PHP devono avere estensione .php;

I nomi dei file non devono contenere spazi;
si consiglia vivamente di scrivere nomi tutto in minuscolo.

Esempio di nomi di file corretti:
prova.htm, logo.gif, chi_siamo.htm, index.php

Esempio di nomi di file non corretti:
prova.html, logo.GIF, chi siamo.htm

Il file index.htm (o anche index.php)
al primo livello della vostra area di lavoro (il cos�detto "web root") deve essere caricato un file di nome index.htm a rappresentare la vostra home page. Se non inserite questo file il tentativo di accedere al vostro sito con indirizzo del tipo http://utenti.garamond.it/identificativo produrr� un errore di Forbidden.

Le cartelle
Nella vostra area di lavoro potete creare nuove cartelle cliccando sull'icona "nuova cartella" . La cartella verr� creata all'interno della cartella in cui in quel momento vi trovate (all'inizio vi trovate nella cartella principale "web root"). Per poter caricare file all'interno della cartella dove prima entrarci e da qui cliccare sull'icona "nuovo file" o sull'icona "nuova cartella" per creare eventuali sottocartelle.
Ad esempio nella cartella principale potete creare una cartella di nome "img" e qui metterci il file "logo.gif".
Nel file "index.htm" per poter riferire quell'immagine dovrete specificare il percorso img/logo.gif.
